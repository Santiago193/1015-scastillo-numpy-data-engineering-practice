# 🛠️ 1015: NumPy Algorithmic Challenges & Data Practice

This repository contains the practical implementation of the **Python & NumPy Workshop**. It demonstrates the application of vectorized logic to solve real-world data engineering problems, focusing on performance and code elegance.

## 📈 Engineering Milestones Achieved

### Data Manipulation & Reshaping
* **Advanced Slicing:** Efficient extraction of sub-matrices and data segments.
* **Boolean Masking:** High-speed data filtering using logical conditions instead of expensive `for` loops.
* **Fancy Indexing:** Non-contiguous data selection for complex dataset sampling.

### Mathematical Pre-processing
* **Feature Scaling:** Manual implementation of **Min-Max Normalization** to transform raw data into a [0, 1] range.
* **Broadcasting Logic:** Executing arithmetic operations across arrays with different shapes by applying NumPy's compatibility rules.
* **Structural Engineering:** Dataset merging via `vstack`/`hstack` and controlled segmentation using `vsplit`.
